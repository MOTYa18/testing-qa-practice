# testing-qa-practice

Учебный проект по автоматизации UI-тестов для сайта qa-practice.com с использованием `Python`, `Pytest` и `Selenium WebDriver`.

В проекте реализован базовый автотестовый фреймворк по паттерну Page Object Model.
Тесты покрывают основные пользовательские сценарии взаимодействия с веб-элементами.

Цель проекта — отработка практических навыков UI-автоматизации:
написание автотестов на Python
работа с Selenium WebDriver
применение паттерна Page Object Model
организация структуры автотестового проекта

Запуск тестов:
pytest
